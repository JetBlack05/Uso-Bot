The bot right here won't run until you installed all of the missing packages !
Here they are : discord.py - oppai0.9.5 (https://github.com/oamaok/oppai-web) - osu api (https://github.com/khazhyk/osuapi)

This project is running under python3.5 with a linux distribution (Raspbian Jessie Light -> https://www.raspberrypi.org/downloads/raspbian/)
