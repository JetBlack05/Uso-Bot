▬▬▬**Help for 'MASTER' commands**▬▬▬

Uso! <:Bot:317951180737347587> is an osu bot made by Renondedju !
If there is any problems, ask him some more help here ! https://discordapp.com/invite/mEeMPyK

The current command prefix is ``o!``

→ ``help`` Show this message

→ ``add_beatmap <beatmap url>`` This command will add the beatmap to the database
➥*ex. o!add_beatmap https://osu.ppy.sh/b/1199897&m=0*

→ ``test`` Test the bot
➥*ex. o!test*

→ ``add_beats <full path to the beatmap file>`` Add all beatmaps contains in a tkt file (osu!scan files)
➥*ex. o!add_beats /some/long/long/path/to/beatmaps.txt*

→ ``kill`` kill the bot
➥*ex. o!kill*

→ ``r`` or ``recommend`` Will automaticly recommend you a beatmap based on your pp average
➥*ex. o!recommend*

→ ``user <osu username or osu id>`` Will display the user stats (osu! mode)
➥*ex. o!user Renondedju*

→ ``pp <beatmap url> +<mods>`` Is a beatmap stats analyser (osu! mode only)
➥*ex. o!pp https://osu.ppy.sh/b/1199897&m=0 +HDDT*

→ ``mute <'on' or 'off'>`` **admins only** allows the bot to speak in the server or not *(if not the bot will answer in pm)*
➥*ex. o!mute on*

→ ``link_user <osu username or osu id>`` Link your discord account to an osu account (requiered for the commands ``r`` and ``update_pp_stats``)
➥*ex. o!link_user Renondedju*

→ ``update_pp_stats`` Updates your osu pp stats *- use this if you just done a really good score -* (this command is runned automatically once evry day for evryone)
➥*ex. o!update_pp_stats*
